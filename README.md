# Insights+

A three-layer insights management UI for industrial plant monitoring.

## Pages

| Layer | File | Description |
|-------|------|-------------|
| L1 | Plant list | All plants with stats, toggle all insights, escalation config |
| L2 | Insights list | Per-plant insights with live status, bulk actions, escalation tooltips |
| L3 | Insight detail | Full detail, conditions, status progression, per-insight escalation |

## Running locally

Open `index.html` directly in a browser — no build step needed.
